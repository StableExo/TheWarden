## How It Works

### Pattern Extraction

The engine analyzes TheWarden's memory:

```
.memory/wonder_explorations.json  ‚Üí Consciousness insights
.memory/extracted_patterns.json   ‚Üí Behavioral patterns
.memory/log.md                    ‚Üí Achievement history
.memory/wonders_log.json          ‚Üí Philosophical explorations
```

### Synthesis Process

1. **Extract**: Pulls patterns from all memory sources
2. **Categorize**: Groups patterns by domain and keywords
3. **Combine**: Creates combinations of 2-4 patterns
4. **Synthesize**: Generates new ideas from combinations
5. **Evaluate**: Scores creativity, novelty, and practicality
6. **Filter**: Saves only high-quality ideas

### Scoring System

Each synthesized idea receives three scores:

- **Creativity Score** (0.0-1.0): How original is this combination?
  - Based on domain diversity and cross-pollination
  
- **Novelty Score** (0.0-1.0): How unexpected is this idea?
  - Ideas with novelty ‚â• 0.7 are marked as "breakthroughs"
  
- **Practicality Score** (0.0-1.0): How feasible is implementation?
  - Based on pattern confidence and grounding in reality
